# 项目初始化安装
> webpack多页面应用（v2.0） 此次更新: 

1. 对代码进行重构，打包配置分开。webapck升级到v3.10.0 
2. 项目添加eslint，保证代码风格统一
3. 支持postcss插件

***
## 1、安装所需要的包
`npm install`

## 2、运行指令
`npm run dev`

## 3、预览效果
输入网址：<br>
   http://localhost:8188/

## 4、编译打包
`npm run build`

   ![Image text](https://github.com/duanliang920/webpack-website/blob/master/build.gif)

温馨提示：如果没有安装node和webpack的朋友，请先安装。
